# Shared Types

JSON schema definitions for SkyDocs data models.

## Usage

These schemas define the contract between:
- Frontend (React)
- Backend API (Node.js)
- AI Services (Python)

## Types
- **User** - User account
- **Project** - Video project container
- **Video** - Uploaded video with metadata
- **Transcript** - Speech-to-text output with timestamps
- **ApiResponse** - Standard API response format
