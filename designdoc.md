Document Type: UI/UX Design SpecificationTheme: Dark Mode / SaaS Dashboard1. Visual Identity & Style GuideColor PaletteThe design uses a deep, dark theme to minimize eye strain, with high-contrast gradients for primary actions.Canvas Background: Deep Midnight Blue/Grey (#0F0F12 approx)Sidebar Background: Near Black (#08080A)Card Background: Dark Violet/Grey (#18181B to #1E1E24)Primary Accent (Brand): Gradient Pink to Purple (#EC4899 $\to$ #8B5CF6) - Used for "New video" button, Upgrade banner, and active states.Text Primary: White (#FFFFFF)Text Secondary: Cool Grey (#9CA3AF)Borders/Dividers: Subtle Dark Grey (#27272A)TypographyFont Family: Modern Sans-Serif (e.g., Inter, Plus Jakarta Sans, or DM Sans).Hierarchy:Headings: Semi-bold (600), ~16px-18px (e.g., "Create a new video").Body: Regular (400), ~14px.Labels/Micro-copy: Medium (500), ~12px (e.g., "AI tools", "NEW" badge).2. Layout ArchitectureThe layout is a standard Sidebar + Main Content grid structure.A. Sidebar (Navigation)Width: Fixed (approx. 240px - 260px).Structure:Header: Logo ("Clueso") with collapse trigger icon.Primary Actions: Large "New video" CTA (Gradient background).Search: Input field with icon.Main Nav Links: Home, All Projects, Video Templates, Auto-update, Team, Analytics. Active state indicates a left border strip in purple.Footer Nav: Settings, Trash.User Context:Upgrade Banner: A standout purple container urging plan upgrades.Profile Section: User Avatar, Team Name, and Email details.B. Main Content AreaPadding: Generous padding (~32px) around the content block.Grid System:Top Row: 3-Column Grid for "Create" actions.Second Row: 3-Column Grid (smaller height) for "AI Tools".Middle Section: Full-width "Empty State" visualization.Bottom Row: Split view. 2-Column Grid for "Getting Started" videos vs. Stacked rows for "Team/Help" actions.3. Component Specifications1. Action Cards (Create a new video)Visuals: Rectangular cards with a subtle gradient or solid dark background.Content: Icon (Left/Top), Title (Bold), Description (Grey text).Hover Effect: Slight lighten of background or border glow.Types:Record screenUpload a videoUpload a slide deck2. Tool Cards (AI Tools)Distinction: Smaller vertical height than Action Cards.Badging: "NEW" badge next to section header.Content: Feature icon, Title, Description.Items: Cuts, Auto-update, Translator.3. Empty State (Recent Projects)Purpose: Placeholder when no data exists.Visual: A concentric circle "radar" or "sonar" graphic centered in the container. Floating file icons scattered on the radar rings.Text: "No projects found" (Heading) + "Choose an option above to create your first project" (Subtext).4. Onboarding & SupportVideo Thumbnails: 16:9 aspect ratio cards with play buttons overlaying a preview image.Action Rows: Wide, horizontal buttons for "Invite team members" and "Help center". High contrast "Add users" / "Learn more" buttons on the right side of these rows.4. Implementation Suggestions (CSS/Tailwind)Gradients: Use bg-gradient-to-r from-pink-500 to-purple-600 for the primary CTA.Glassmorphism: The "Empty State" radar effect could utilize subtle opacity layers (opacity-10, opacity-20) to create depth.Borders: Use border border-white/10 for card outlines to ensure they are visible against the dark background without being harsh.
